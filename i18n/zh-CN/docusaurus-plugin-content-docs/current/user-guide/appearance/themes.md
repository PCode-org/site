# 主题

PCode 支持多种主题，让您可以选择最适合的界面外观。

## 功能概述

主题功能提供了以下功能：

- **浅色主题**：明亮清爽的浅色界面
- **深色主题**：护眼的深色界面
- **系统主题**：自动跟随系统主题设置
- **自定义主题**：自定义颜色和样式

## 可用主题

### 浅色主题

明亮清爽的界面主题：

- **适合场景**：光线充足的环境
- **特点**：清晰易读，色彩鲜明
- **推荐**：白天使用

### 深色主题

护眼的深色界面主题：

- **适合场景**：光线较暗的环境
- **特点**：减少眼疲劳，节省电量（OLED 屏幕）
- **推荐**：夜间使用

### 系统主题

自动跟随系统的主题设置：

- **自动切换**：根据系统设置自动切换
- **无缝体验**：与系统主题保持一致
- **推荐**：希望自动跟随系统的用户

## 切换主题

### 快速切换

使用主题切换按钮快速切换：

1. 点击界面右上角的主题切换图标
2. 选择要切换的主题
3. 界面立即应用新主题

### 键盘快捷键

使用快捷键快速切换主题：

- `Ctrl + Shift + T`：切换浅色/深色主题
- `Ctrl + Shift + S`：切换到系统主题

### 设置中切换

在设置中切换主题：

1. 打开设置
2. 找到 **外观** 设置
3. 选择要使用的主题
4. 保存设置

## 主题应用范围

主题影响以下界面元素：

- **主界面**：主界面的背景和前景色
- **面板**：各个面板的颜色
- **按钮**：按钮的颜色和样式
- **输入框**：输入框的样式
- **消息**：消息的颜色和样式
- **代码块**：代码块的高亮样式
- **图表**：图表的颜色方案

## 主题预览

在切换主题前可以预览：

- **实时预览**：鼠标悬停时预览
- **快速切换**：点击即可应用
- **即时生效**：无需重启应用

## 自定义主题

### 颜色自定义

自定义主题的颜色：

- **主色调**：主要交互元素的颜色
- **背景色**：界面背景的颜色
- **文字色**：文字的颜色
- **边框色**：边框和分隔线的颜色

### 样式自定义

自定义主题的样式：

- **圆角**：元素的圆角大小
- **阴影**：阴影效果
- **字体**：界面的字体
- **间距**：元素间距

### 保存自定义主题

保存自定义的主题：

1. 完成自定义设置
2. 点击 **保存为主题**
3. 输入主题名称
4. 保存主题

## 主题同步

### 跨设备同步

如果启用了云端同步：

- **自动同步**：主题设置自动同步到云端
- **跨设备一致**：在不同设备上保持一致
- **实时更新**：更改后立即同步

### 导入导出主题

导入和导出自定义主题：

- **导出主题**：将自定义主题导出为文件
- **导入主题**：从文件导入主题
- **分享主题**：与他人分享自定义主题

## 主题和代码高亮

### 代码高亮主题

代码块的高亮主题会随界面主题自动调整：

- **浅色主题**：使用浅色代码高亮主题
- **深色主题**：使用深色代码高亮主题
- **独立设置**：可以独立设置代码高亮主题

### 支持的高亮主题

PCode 支持多种代码高亮主题：

- VS Code Dark
- VS Code Light
- GitHub Dark
- GitHub Light
- Monokai
- Dracula
- Solarized Dark
- Solarized Light

## 主题性能

### 渲染性能

不同主题的渲染性能：

- **浅色主题**：通常渲染速度较快
- **深色主题**：在 OLED 屏幕上更省电
- **透明效果**：透明效果可能影响性能

### 动画效果

主题切换的动画效果：

- **渐变过渡**：平滑的颜色渐变
- **动画时长**：可配置动画时长
- **禁用动画**：可以禁用动画以提升性能

## 最佳实践

### 选择主题

- **环境光线**：根据环境光线选择主题
- **个人偏好**：根据个人偏好选择
- **使用场景**：根据使用场景选择

### 眼睛健康

- **定期休息**：无论使用什么主题，都要定期休息
- **亮度调节**：配合屏幕亮度调节
- **蓝光过滤**：配合蓝光过滤功能

### 一致性

- **保持一致**：在同一场景下使用相同的主题
- **减少切换**：避免频繁切换主题
- **习惯养成**：养成使用固定主题的习惯

## 故障排除

### 主题不生效

如果主题切换不生效：

1. **刷新页面**：尝试刷新页面
2. **清除缓存**：清除浏览器缓存
3. **检查设置**：检查主题设置是否正确
4. **重启应用**：尝试重启应用

### 颜色显示异常

如果颜色显示异常：

1. **检查系统**：检查系统的显示设置
2. **更新浏览器**：更新浏览器到最新版本
3. **重置主题**：重置为默认主题
4. **联系支持**：如果问题持续，联系技术支持
