# 语言

PCode 支持多种语言，让您可以使用熟悉的语言进行交互。

## 功能概述

语言功能提供了以下功能：

- **多语言支持**：支持多种界面语言
- **实时切换**：实时切换界面语言
- **自动检测**：自动检测系统语言
- **翻译贡献**：参与翻译贡献

## 支持的语言

PCode 目前支持以下语言：

- **简体中文 (zh-CN)**：简体中文
- **English (en)**：英语

:::info 计划中

我们正在添加更多语言支持。如果您想帮助翻译，请考虑参与翻译贡献。

:::

## 切换语言

### 快速切换

使用语言切换器快速切换：

1. 点击界面右上角的语言图标
2. 选择要使用的语言
3. 界面立即切换到新语言

### 设置中切换

在设置中切换语言：

1. 打开设置
2. 找到 **语言** 设置
3. 选择要使用的语言
4. 保存设置

## 语言应用范围

语言设置影响以下内容：

- **界面文本**：所有界面元素
- **按钮和标签**：按钮和标签的文本
- **消息和提示**：系统消息和提示
- **帮助文档**：内置的帮助文档
- **错误消息**：错误和警告消息

## 不受影响的内容

以下内容不受语言设置影响：

- **AI 回复内容**：AI 回复使用对话中使用的语言
- **代码内容**：代码保持原样
- **用户输入**：用户输入的内容
- **项目文件**：项目中的文件内容

## 自动检测

PCode 可以自动检测系统语言：

- **首次使用**：首次使用时自动检测
- **系统语言**：使用操作系统的语言设置
- **浏览器语言**：使用浏览器的语言设置
- **手动覆盖**：可以手动覆盖自动检测的结果

## 翻译质量

### 官方翻译

PCode 的官方翻译：

- **专业翻译**：由专业翻译人员翻译
- **社区审核**：由社区审核和改进
- **持续更新**：持续更新和改进

### 机器翻译

部分内容可能使用机器翻译：

- **临时翻译**：新功能的临时翻译
- **辅助翻译**：辅助人工翻译
- **标记清晰**：明确标记机器翻译内容

## 日期和时间格式

语言设置也会影响日期和时间的显示格式：

- **中文**：`2024年1月8日`
- **English**：`January 8, 2024`

## 数字格式

数字格式也会随语言设置变化：

- **中文**：`1,234.56`
- **English**：`1,234.56`

## 翻译贡献

### 如何贡献

您可以帮助改进 PCode 的翻译：

1. 访问 PCode 翻译平台
2. 注册或登录
3. 选择您要翻译的语言
4. 开始翻译或审核

### 翻译指南

翻译时请遵循以下指南：

- **保持简洁**：UI 文本应简洁明了
- **一致性**：保持术语翻译的一致性
- **本地化**：考虑文化和语言的差异
- **测试**：翻译后在实际界面中测试

### 审核流程

翻译的审核流程：

1. **提交翻译**：提交您的翻译
2. **社区审核**：其他用户审核您的翻译
3. **管理员批准**：管理员批准后合并
4. **发布更新**：在下一版本中发布

## 语言文件

### 文件位置

语言文件位于以下位置：

- **应用内**：应用内置的语言文件
- **云端**：云端更新的语言文件
- **自定义**：用户自定义的语言文件

### 文件格式

语言文件使用标准格式：

- **JSON**：易于编辑和解析
- **UTF-8**：支持所有 Unicode 字符
- **结构化**：按模块和功能组织

## 自定义翻译

### 创建自定义翻译

您可以创建自定义翻译：

1. 导出现有的语言文件
2. 编辑翻译内容
3. 导入自定义翻译
4. 应用自定义翻译

:::caution 注意

自定义翻译不会自动更新，需要在官方翻译更新后手动同步。

:::

## 最佳实践

### 选择语言

- **母语优先**：优先选择您的母语
- **工作语言**：考虑您的工作语言
- **团队一致**：与团队使用相同的语言

### 切换语言

- **重要操作**：避免在重要操作中切换语言
- **保存工作**：切换前确保保存工作
- **刷新页面**：切换后刷新页面确保完全应用

### 翻译贡献

- **母语贡献**：优先贡献您的母语翻译
- **质量优先**：质量比数量更重要
- **持续改进**：持续改进和维护翻译

## 故障排除

### 语言不生效

如果语言切换不生效：

1. **刷新页面**：尝试刷新页面
2. **清除缓存**：清除浏览器缓存
3. **检查设置**：检查语言设置是否正确
4. **重启应用**：尝试重启应用

### 翻译错误

如果发现翻译错误：

- **报告问题**：通过适当渠道报告
- **贡献修复**：直接贡献翻译修复
- **临时方案**：使用英语作为临时方案

### 缺失翻译

如果某些文本未被翻译：

- **正常情况**：新功能可能暂时缺失翻译
- **等待更新**：等待下一版本更新
- **贡献翻译**：贡献缺失的翻译
