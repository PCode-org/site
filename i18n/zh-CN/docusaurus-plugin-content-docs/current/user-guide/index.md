# 用户指南

欢迎使用 PCode 用户指南。本指南将帮助您快速上手并充分利用 PCode 的功能。

## 快速开始

如果您是第一次使用 PCode，建议按以下顺序阅读：

1. **[外观 - 主题](./appearance/themes.md)** - 选择适合您的界面主题
2. **[外观 - 语言](./appearance/languages.md)** - 选择您的首选语言
3. **[设置概述](./settings/overview.md)** - 了解可用的设置选项

## 主要功能

### 外观

自定义 PCode 的外观：

- **[主题](./appearance/themes.md)** - 切换界面主题
- **[语言](./appearance/languages.md)** - 切换界面语言

### 设置

管理 PCode 设置：

- **[设置概述](./settings/overview.md)** - 查看所有设置选项
