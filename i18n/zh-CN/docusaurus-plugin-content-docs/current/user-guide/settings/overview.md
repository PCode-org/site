# 设置概述

设置是管理 PCode 行为和偏好的中心位置。

## 功能概述

设置提供了以下功能：

- **全局配置**：配置全局的应用设置
- **项目配置**：配置项目特定的设置
- **偏好管理**：管理个人偏好
- **账户管理**：管理账户信息

## 打开设置

您可以通过以下方式打开设置：

1. 点击界面右上角的 **设置** 图标
2. 使用键盘快捷键 `Ctrl + ,`
3. 从主菜单选择 **设置**

## 设置类别

设置按类别组织：

### 外观

配置界面的外观和主题：

- **主题**：选择浅色、深色或系统主题
- **语言**：选择界面语言
- **字体**：配置界面字体
- **布局**：配置界面布局

### 通知

配置各种通知：

- **桌面通知**：配置系统通知
- **会话通知**：配置会话相关通知
- **提案通知**：配置提案相关通知
- **免打扰**：配置免打扰时段

### 项目

配置项目相关的设置：

- **项目范围**：配置项目的可见性
- **项目默认**：设置新项目的默认配置
- **项目同步**：配置项目同步选项

### 会话

配置会话相关的设置：

- **并发限制**：设置最大并发会话数
- **会话默认**：设置新会话的默认配置
- **会话历史**：配置会话历史保留

### 编辑器

配置编辑器相关的设置：

- **代码高亮**：配置代码高亮主题
- **字体大小**：配置编辑器字体大小
- **行号**：显示或隐藏行号
- **自动完成**：配置自动完成选项

### 隐私

配置隐私相关的设置：

- **数据收集**：配置数据收集选项
- **数据共享**：配置数据共享选项
- **本地存储**：配置本地数据存储
- **清除数据**：清除本地数据

### 账户

管理您的账户：

- **账户信息**：查看和编辑账户信息
- **密码**：修改密码
- **API 密钥**：管理 API 密钥
- **订阅**：管理订阅和计费

### 集成

配置第三方服务集成：

- **Git**：配置 Git 集成
- **CI/CD**：配置持续集成
- **通知服务**：配置通知服务集成
- **其他服务**：配置其他服务集成

## 设置搜索

### 搜索框

设置顶部有搜索框：

- **实时搜索**：输入时实时过滤设置
- **模糊匹配**：支持模糊搜索
- **快捷导航**：按 Enter 快速跳转

### 搜索技巧

- **关键词搜索**：使用设置的关键词
- **类别搜索**：按类别名称搜索
- **值搜索**：按设置的值搜索

## 设置同步

### 本地同步

设置在本地自动同步：

- **实时保存**：更改实时保存
- **跨会话**：在不同会话间同步
- **持久化**：在应用重启后保持

### 云端同步

如果启用了云端同步：

- **自动上传**：设置自动上传到云端
- **自动下载**：在新设备上自动下载
- **冲突解决**：自动解决设置冲突

## 设置导入导出

### 导出设置

1. 打开设置
2. 点击 **导出设置**
3. 选择要导出的设置类别
4. 保存设置文件

### 导入设置

1. 打开设置
2. 点击 **导入设置**
3. 选择设置文件
4. 确认导入

:::caution 注意

导入设置将覆盖现有设置，建议先导出备份。

:::

## 重置设置

### 重置所有设置

将所有设置恢复为默认值：

1. 打开设置
2. 滚动到底部
3. 点击 **重置所有设置**
4. 确认重置操作

:::danger 警告

重置所有设置将清除所有自定义配置，此操作不可撤销。

:::

### 重置特定类别

重置特定类别的设置：

1. 打开设置
2. 找到要重置的设置类别
3. 点击该类别的 **重置** 按钮
4. 确认重置操作

## 设置文件

### 文件位置

设置保存在以下位置：

- **用户设置**：`~/.pcode/settings.json`
- **项目设置**：`.pcode/project-settings.json`
- **会话设置**：保存在会话数据中

### 文件格式

设置文件使用 JSON 格式：

- **易于编辑**：可以使用文本编辑器编辑
- **版本控制**：可以纳入版本控制
- **备份恢复**：方便备份和恢复

## 高级设置

### 开发者选项

启用开发者选项可以访问：

- **调试模式**：启用调试功能
- **实验性功能**：启用实验性功能
- **性能监控**：查看性能数据
- **日志级别**：配置日志级别

### 实验性功能

实验性功能是正在开发的功能：

- **功能列表**：显示可用的实验性功能
- **功能描述**：每个功能的描述
- **启用/禁用**：切换功能状态

:::warning 实验性功能

实验性功能可能不稳定，使用时请注意备份重要数据。

:::

## 最佳实践

### 配置管理

- **定期备份**：定期导出设置备份
- **版本控制**：将设置文件纳入版本控制
- **文档记录**：记录重要的设置更改

### 配置优化

- **按需配置**：只配置需要的选项
- **保持简洁**：避免过度复杂的配置
- **定期审查**：定期审查和更新设置

### 团队协作

- **共享配置**：与团队共享配置文件
- **配置规范**：建立团队配置规范
- **文档说明**：为配置提供文档说明

## 键盘快捷键

设置界面中的快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + ,` | 打开设置 |
| `Ctrl + F` | 搜索设置 |
| `Esc` | 关闭设置 |
| `Ctrl + S` | 保存设置 |
