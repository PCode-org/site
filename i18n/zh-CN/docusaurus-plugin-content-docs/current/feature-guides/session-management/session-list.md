# 会话列表

会话列表是 PCode 的核心导航功能，让您可以轻松管理和访问所有会话。

## 功能概述

会话列表提供了以下功能：

- **会话展示**：显示所有会话的标题、描述、状态和时间信息
- **快速切换**：点击会话即可快速切换到该会话的详情页面
- **状态标识**：通过颜色和图标清晰显示会话的当前状态
- **类型区分**：区分聊天会话和提案会话
- **右键菜单**：提供快速操作选项，如在 VS Code 中打开

## 会话状态

会话列表通过不同的徽章（Badge）显示会话的当前状态：

| 状态 | 说明 | 外观 |
|------|------|------|
| 新建 (New) | 会话刚创建，正在初始化 | 蓝色边框 |
| 优化中 (Optimizing) | 正在优化会话配置 | 紫色边框，带旋转图标 |
| 草稿 (Drafted) | 会话内容正在草拟阶段 | 黄色边框 |
| OpenSpec 处理中 | 正在生成 OpenSpec 规范 | 靛蓝色边框，带旋转图标 |
| 审查中 (Reviewing) | 会话内容正在审查 | 青色边框 |
| 执行中 (Executing) | 正在执行会话任务 | 橙色边框，带旋转图标 |
| 执行完成 (Execution Completed) | 任务执行完成 | 绿色边框 |
| 归档中 (Archiving) | 正在归档会话 | 灰色边框，带旋转图标 |
| 已归档 (Archived) | 会话已归档 | 灰色边框 |
| 完成 (Completed) | 会话完全完成 | 翠绿色边框 |

## 会话类型

### 聊天会话

- 蓝色徽章标识
- 用于日常对话和交互
- 支持实时消息流

### 提案会话

- 绿色徽章标识
- 用于 OpenSpec 提案创建和管理
- 包含提案特定的功能和视图

## 创建新会话

会话列表顶部提供了创建按钮：

1. 点击 **聊天** 按钮创建新的聊天会话
2. 点击 **提案** 按钮创建新的提案会话

:::info 提示

创建会话前，请确保已选择一个项目。

:::

## 会话信息展示

每个会话卡片显示以下信息：

- **会话编号**：`#123` 格式的唯一标识
- **会话标题**：会话的主要标识
- **描述**：会话的简要描述（如果有）
- **项目标识**：在全局视图下，显示会话所属项目的缩写
- **最后活动时间**：相对时间显示（如"5分钟前"）
- **会话类型**：聊天或提案
- **当前状态**：会话的处理状态

## 活跃会话指示

当前选中的会话会高亮显示：
- 蓝色边框和背景色
- 主色调环形边框

正在运行的会话会有特殊的动画效果。

## 右键菜单

在会话上右键单击可以打开上下文菜单，提供以下操作：

- **在 VS Code 中打开**：在 VS Code 中打开会话对应的工作区
- **打开仓库**：在浏览器中打开项目的 Git 仓库

## 活跃会话监控

当启用了电源指示器设置时，会话列表顶部会显示当前活跃会话的数量。这可以帮助您了解系统资源的使用情况。

## 排序和分组

会话列表自动按照以下规则排序：

1. 活跃会话优先
2. 按最后活动时间降序
3. 按状态分组
