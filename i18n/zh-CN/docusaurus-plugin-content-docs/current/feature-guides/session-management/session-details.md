# 会话详情

会话详情面板展示了会话的完整信息，包括基本信息和详细属性。

## 功能概述

会话详情面板提供了以下功能：

- **信息查看**：查看会话的标题、描述、状态等基本信息
- **信息编辑**：编辑会话的标题和描述
- **状态监控**：实时监控会话的处理状态
- **提案会话**：为提案类型的会话提供专门的视图

## 标签页

会话详情面板包含两个主要标签页：

### 聊天 (Chat)

显示会话的对话内容和消息流。详见 [会话聊天](./session-chat.md)。

### 信息 (Info)

显示和编辑会话的基本信息：

- **会话标题**：会话的名称
- **会话描述**：会话的详细描述
- **创建时间**：会话创建的时间戳
- **会话状态**：会话的当前处理状态
- **会话类型**：聊天或提案
- **所属项目**：会话关联的项目

## 编辑会话信息

1. 在信息标签页中点击 **编辑** 按钮
2. 修改标题或描述内容
3. 点击 **保存** 提交更改，或 **丢弃** 取消编辑

### 验证规则

- **标题**：
  - 必填字段
  - 最长 200 个字符
- **描述**：
  - 可选字段
  - 最长 14,000 个字符

## 提案会话详情

对于提案类型的会话，详情面板会提供特殊的功能：

- **提案信息**：显示提案的标题、主要诉求等
- **工作区信息**：显示独立工作区的状态
- **OpenSpec 文件**：查看和编辑提案相关的 OpenSpec 规范文件
- **提案图表**：查看提案的流程图和架构图
- **注释功能**：查看和管理提案的注释
- **审查功能**：进行提案审查

## 会话生命周期

会话详情面板反映了会话的完整生命周期：

1. **新建 (New)**：会话刚创建
2. **优化中 (Optimizing)**：正在优化配置
3. **草稿 (Drafted)**：内容草拟中
4. **OpenSpec 处理中**：生成规范中
5. **审查中 (Reviewing)**：内容审查中
6. **执行中 (Executing)**：任务执行中
7. **执行完成**：执行完成
8. **归档中 (Archiving)**：正在归档
9. **已归档 (Archived)**：已归档
10. **完成 (Completed)**：完全完成

## 错误处理

如果会话加载失败，详情面板会显示错误信息，并提供 **重试** 按钮重新加载。

## 自动刷新

会话详情会在以下情况自动刷新：

- 会话状态发生变化
- 提案名称被自动生成
- 会话被归档
- 会话进入等待执行状态
