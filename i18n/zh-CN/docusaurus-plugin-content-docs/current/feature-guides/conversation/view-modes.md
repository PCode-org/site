# 视图模式

视图模式功能允许您切换不同的消息显示方式，以适应不同的使用场景和偏好。

## 功能概述

视图模式提供了以下功能：

- **多种视图**：提供多种预设的视图模式
- **自定义视图**：根据偏好自定义视图设置
- **快速切换**：方便地在不同视图间切换
- **视图保存**：保存视图偏好设置

## 可用视图模式

### 标准视图

默认的聊天界面布局：

- **消息列表**：按时间顺序显示所有消息
- **输入框**：底部固定输入框
- **侧边栏**：显示会话信息
- **工具面板**：显示可用工具

**适用场景**：
- 日常对话
- 一般的 AI 交互
- 首次使用 PCode

### 专注视图

隐藏辅助信息，专注对话内容：

- **简化界面**：隐藏侧边栏和工具面板
- **更大显示区域**：消息显示区域最大化
- **最小化干扰**：减少视觉干扰元素

**适用场景**：
- 阅读长篇 AI 回复
- 深度思考对话内容
- 演示和展示

### 详细视图

显示更多的元数据和调试信息：

- **时间戳**：每条消息的精确时间
- **消息 ID**：消息的唯一标识符
- **模型信息**：使用的 AI 模型
- **Token 使用**：Token 使用统计
- **调试信息**：技术调试信息

**适用场景**：
- 调试问题
- 分析性能
- 技术分析

### 分屏视图

将界面分为多个独立面板：

- **消息列表**：左侧显示消息列表
- **详情面板**：右侧显示选中消息的详情
- **工具调用**：独立的工具调用面板
- **待办任务**：独立的任务面板

**适用场景**：
- 多任务处理
- 同时查看多个信息源
- 大屏幕使用

### 紧凑视图

压缩显示内容，显示更多信息：

- **较小字体**：使用较小的字体
- **紧凑间距**：减少元素间距
- **更多内容**：在屏幕中显示更多消息

**适用场景**：
- 浏览大量历史消息
- 搜索特定内容
- 小屏幕使用

## 视图切换

### 视图切换器

使用视图切换器快速切换视图：

- **下拉菜单**：选择视图模式
- **快捷键**：使用快捷键切换
- **视图图标**：点击视图图标

### 视图切换按钮

在聊天面板顶部有视图切换按钮：

- **标准视图**：显示为默认图标
- **专注视图**：显示为书本图标
- **详细视图**：显示为代码图标
- **分屏视图**：显示为分屏图标
- **紧凑视图**：显示为压缩图标

## 自定义视图设置

### 消息显示

- **字体大小**：调整消息字体大小
- **行高**：调整行高
- **最大宽度**：设置消息最大宽度
- **对齐方式**：选择消息对齐方式

### 元数据显示

- **时间戳**：显示/隐藏时间戳
- **用户名**：显示/隐藏用户名
- **消息 ID**：显示/隐藏消息 ID
- **Token 数**：显示/隐藏 Token 使用数

### 面板显示

- **侧边栏**：显示/隐藏侧边栏
- **工具面板**：显示/隐藏工具面板
- **任务面板**：显示/隐藏任务面板
- **统计面板**：显示/隐藏统计面板

## 视图保存

### 自动保存

视图偏好设置自动保存：

- **持久化**：设置在浏览器中持久化
- **跨会话**：设置在会话间保持
- **云端同步**：可选的云端同步

### 预设视图

保存自定义的视图预设：

- **创建预设**：保存当前视图为预设
- **命名预设**：为预设命名
- **快速应用**：一键应用预设

## 响应式设计

视图模式自动适应不同屏幕尺寸：

- **桌面**：完整功能和多面板布局
- **平板**：优化的两面板布局
- **手机**：单面板垂直布局
- **自动调整**：根据屏幕尺寸自动调整

## 辅助功能

### 高对比度

支持高对比度模式：

- **增强对比度**：提高元素间的对比度
- **视觉清晰**：更清晰的视觉层次
- **可访问性**：改善可访问性

### 大字体

支持大字体模式：

- **放大显示**：放大所有文本
- **保持布局**：保持布局的完整性
- **易于阅读**：提高可读性

## 键盘快捷键

使用快捷键快速切换视图：

| 快捷键 | 视图模式 |
|--------|----------|
| `Ctrl + 1` | 标准视图 |
| `Ctrl + 2` | 专注视图 |
| `Ctrl + 3` | 详细视图 |
| `Ctrl + 4` | 分屏视图 |
| `Ctrl + 5` | 紧凑视图 |

## 最佳实践

### 选择视图模式

- **日常使用**：使用标准视图
- **阅读长内容**：使用专注视图
- **调试问题**：使用详细视图
- **多任务处理**：使用分屏视图
- **浏览历史**：使用紧凑视图

### 自定义视图

- 根据工作流程自定义视图
- 保存常用的视图配置为预设
- 尝试不同的视图找到最适合的
