# 项目详情

项目详情面板显示项目的完整信息，包括基本信息、统计信息和相关配置。

## 功能概述

项目详情面板提供了以下功能：

- **基本信息**：显示项目的基本属性
- **统计信息**：显示项目的使用统计
- **会话列表**：显示项目相关的所有会话
- **配置管理**：管理项目的配置设置
- **Markdown 标签页**：查看项目相关的 Markdown 文档

## 项目信息

### 基本信息

- **项目名称**：项目的主要标识
- **项目描述**：项目的详细描述
- **项目路径**：项目在文件系统中的完整路径
- **项目 ID**：项目的唯一标识符
- **创建时间**：项目的创建时间
- **最后更新**：项目最后更新的时间

### 项目配置

- **项目类型**：本地项目或远程项目
- **仓库 URL**：远程仓库的地址（如果适用）
- **分支**：当前 Git 分支
- **项目范围**：项目的可见性范围

## 统计信息

### 会话统计

- **总会话数**：项目中的所有会话数量
- **活跃会话数**：当前活跃的会话数量
- **已归档会话数**：已归档的会话数量
- **会话类型分布**：聊天会话和提案会话的数量

### 活动统计

- **总消息数**：项目中所有会话的消息总数
- **工具调用数**：执行的工具调用总数
- **最后活动**：项目最后活动的时间
- **活动频率**：项目使用的频率

## 标签页

### 概览标签页

显示项目的概览信息：

- 项目基本信息
- 快速统计
- 最近活动
- 快速操作

### 会话标签页

显示项目相关的所有会话：

- 会话列表
- 会话状态
- 会话类型
- 快速跳转到会话

### Markdown 标签页

显示项目相关的 Markdown 文档：

- **README.md**：项目的 README 文件
- **其他文档**：项目中的其他 Markdown 文件
- **实时预览**：Markdown 的实时渲染预览
- **编辑功能**：直接编辑 Markdown 文件

### SDD 标签页

显示软件设计文档（SDD）：

- **SDD 文件**：项目中的 SDD 文件
- **SDD 差异**：查看 SDD 的变更
- **版本对比**：比较不同版本的 SDD

## 项目操作

### 编辑项目信息

1. 点击 **编辑** 按钮
2. 修改项目信息
3. 保存更改

### 刷新项目

点击 **刷新** 按钮可以：

- 更新项目信息
- 同步远程仓库
- 重新加载统计数据

### 删除项目

- 点击 **删除** 按钮
- 确认删除操作
- **注意**：删除项目不会删除文件系统中的文件

## Git 集成

### 仓库信息

- **仓库 URL**：显示远程仓库地址
- **当前分支**：显示当前 Git 分支
- **提交信息**：显示最近的提交信息

### Git 操作

- **拉取**：从远程仓库拉取更改
- **推送**：推送本地更改到远程
- **查看日志**：查看 Git 提交日志
- **打开仓库**：在浏览器中打开远程仓库

## 项目路径

### 路径显示

- **完整路径**：显示项目的完整路径
- **路径复制**：一键复制项目路径
- **路径验证**：验证路径是否有效

### 路径操作

- **在文件管理器中打开**：打开项目的文件位置
- **在终端中打开**：在终端中打开项目路径
- **在 VS Code 中打开**：在 VS Code 中打开项目

## 项目成员

如果项目支持多用户：

- **成员列表**：显示项目成员
- **成员角色**：显示成员的角色
- **添加成员**：邀请新成员加入项目
- **管理权限**：管理成员的权限
