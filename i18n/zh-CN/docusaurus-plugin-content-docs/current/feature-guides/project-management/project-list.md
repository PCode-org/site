# 项目列表

项目列表是管理 PCode 项目的中心位置，您可以创建、查看和管理所有项目。

## 功能概述

项目列表提供了以下功能：

- **项目浏览**：查看所有项目的列表
- **快速切换**：快速切换到不同的项目
- **项目创建**：创建新项目
- **项目信息**：查看项目的基本信息
- **项目操作**：对项目执行各种操作

## 项目信息显示

每个项目卡片显示以下信息：

- **项目名称**：项目的主要标识
- **项目描述**：项目的简要描述（如果有）
- **项目路径**：项目在文件系统中的路径
- **会话数量**：项目中包含的会话数
- **最后活动**：项目最后活动的时间
- **项目标识**：项目的颜色标识和缩写

## 项目类型

### 本地项目

- 存储在本地文件系统
- 直接访问文件和目录
- 完整的文件操作权限

### 远程项目

- 连接到远程 Git 仓库
- 需要克隆或拉取
- 支持远程协作

## 创建项目

### 创建新项目

1. 点击项目列表中的 **创建项目** 按钮
2. 填写项目信息：
   - 项目名称
   - 项目描述（可选）
   - 项目路径
3. 选择项目类型
4. 点击 **创建**

### 克隆项目

对于远程项目：

1. 点击 **克隆项目** 按钮
2. 输入远程仓库 URL
3. 选择本地路径
4. 点击 **克隆**

## 项目操作

### 切换项目

点击项目卡片可以切换到该项目：

- 当前项目会高亮显示
- 会话列表会更新为该项目的会话
- 所有操作会在选定项目的上下文中进行

### 项目设置

点击项目设置图标可以：

- 编辑项目信息
- 配置项目范围
- 管理项目成员
- 设置项目权限

### 删除项目

- 点击删除图标
- 确认删除操作
- **注意**：删除项目不会删除文件系统中的文件

## 项目视图模式

### 卡片视图

以卡片形式显示项目：

- 更大的项目信息显示
- 适合项目数量较少的情况
- 更好的视觉展示

### 列表视图

以列表形式显示项目：

- 紧凑的显示方式
- 适合项目数量较多的情况
- 更易于快速浏览

### 表格视图

以表格形式显示项目：

- 结构化的数据显示
- 支持排序和筛选
- 适合大量项目管理

## 项目排序

项目列表支持多种排序方式：

- **按名称**：按项目名称字母顺序
- **按创建时间**：按项目创建时间
- **按最后活动**：按最后活动时间
- **按会话数量**：按会话数量排序

## 项目筛选

### 按类型筛选

- 显示所有项目
- 仅显示本地项目
- 仅显示远程项目

### 按状态筛选

- 活跃项目
- 归档项目
- 所有项目

### 搜索项目

使用搜索框快速查找项目：

- 按项目名称搜索
- 按项目描述搜索
- 实时搜索结果

## 项目标识

### 颜色标识

每个项目有独特的颜色标识：

- **自动生成**：根据项目名称自动生成
- **手动设置**：可以手动指定
- **视觉区分**：帮助快速识别项目

### 项目缩写

项目缩写的生成规则：

- 取项目名称的前两个字符
- 或者取每个单词的首字母
- 用于紧凑显示

## 项目范围

项目范围决定了项目在界面中的可见性：

- **全局范围**：项目在全局视图中可见
- **项目范围**：项目仅在项目视图中可见
- **配置**：在项目设置中配置
