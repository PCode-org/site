# 配置面板

配置面板是管理 PCode 设置和偏好的中心位置。

## 功能概述

配置面板提供了以下功能：

- **项目范围设置**：配置项目的可见性范围
- **通知设置**：管理各种通知偏好
- **电源指示器设置**：配置活跃会话指示器
- **注释编辑器**：编辑和管理注释模板
- **全局配置**：配置全局应用设置

## 打开配置面板

您可以通过以下方式打开配置面板：

1. 点击界面右上角的 **设置** 图标
2. 使用键盘快捷键 `Ctrl + ,`
3. 从主菜单选择 **设置**

## 配置选项

### 项目范围设置

配置项目在界面中的可见性：

- **全局范围**：项目在全局视图中可见
- **项目范围**：项目仅在项目视图中可见

**使用场景**：
- 隐藏不常用项目
- 组织工作空间
- 减少界面干扰

### 通知设置

配置各种通知的偏好：

#### 桌面通知

- **启用桌面通知**：允许显示系统通知
- **通知类型**：选择要接收的通知类型
- **免打扰时段**：设置不接收通知的时段

#### 会话通知

- **新会话通知**：新会话创建时通知
- **会话更新通知**：会话状态变化时通知
- **会话完成通知**：会话完成时通知

#### 提案通知

- **提案创建通知**：新提案创建时通知
- **提案审查通知**：提案需要审查时通知
- **提案执行通知**：提案执行状态变化时通知

#### 评论通知

- **新评论通知**：收到新评论时通知
- **回复通知**：评论被回复时通知
- **提及通知**：被提及时通知

### 电源指示器设置

配置活跃会话指示器的显示：

#### 显示设置

- **显示指示器**：显示/隐藏电源指示器
- **显示位置**：选择指示器显示位置
- **显示样式**：选择指示器样式

#### 颜色设置

- **低负载颜色**：设置低负载时的颜色（默认绿色）
- **中等负载颜色**：设置中等负载时的颜色（默认黄色）
- **高负载颜色**：设置高负载时的颜色（默认红色）

#### 阈值设置

- **警告阈值**：设置显示警告的并发数
- **危险阈值**：设置显示危险的并发数

### 注释编辑器

管理和编辑注释模板：

#### 模板管理

- **创建模板**：创建新的注释模板
- **编辑模板**：编辑现有模板
- **删除模板**：删除不需要的模板

#### 模板内容

- **模板名称**：模板的标识名称
- **模板内容**：模板的文本内容
- **变量支持**：支持动态变量插入

#### 模板使用

- **快速插入**：在添加注释时快速插入模板
- **变量替换**：自动替换模板中的变量
- **自定义编辑**：插入后可以继续编辑

## 配置文件

配置保存在以下位置：

- **用户配置**：`~/.pcode/config.json`
- **项目配置**：`.pcode/project.json`
- **会话配置**：自动保存在会话数据中

### 配置优先级

配置按以下优先级应用：

1. **会话配置**：当前会话的配置
2. **项目配置**：当前项目的配置
3. **用户配置**：用户的全局配置
4. **默认配置**：系统的默认配置

## 配置同步

### 本地同步

配置在本地自动同步：

- **实时保存**：配置更改实时保存
- **跨会话**：配置在不同会话间同步
- **持久化**：配置在应用重启后保持

### 云端同步（可选）

如果启用了云端同步：

- **自动上传**：配置自动上传到云端
- **自动下载**：在新设备上自动下载配置
- **冲突解决**：自动解决配置冲突

## 重置配置

### 重置所有设置

将所有配置恢复为默认值：

1. 打开配置面板
2. 滚动到底部
3. 点击 **重置所有设置**
4. 确认重置操作

:::danger 警告

重置所有设置将清除所有自定义配置，此操作不可撤销。

:::

### 重置特定设置

重置特定类别的设置：

1. 打开配置面板
2. 找到要重置的设置类别
3. 点击该类别的 **重置** 按钮
4. 确认重置操作

## 配置搜索

### 搜索框

配置面板顶部有搜索框：

- **实时搜索**：输入时实时过滤设置
- **高亮匹配**：高亮显示匹配的设置
- **快捷导航**：按 Enter 快速跳转到匹配项

### 搜索技巧

- **关键词搜索**：使用设置的关键词搜索
- **类别搜索**：按类别名称搜索
- **值搜索**：按设置的值搜索

## 配置导入导出

### 导出配置

1. 打开配置面板
2. 点击 **导出配置** 按钮
3. 选择要导出的配置类别
4. 保存配置文件

### 导入配置

1. 打开配置面板
2. 点击 **导入配置** 按钮
3. 选择配置文件
4. 确认导入

:::info 注意

导入配置将覆盖现有配置，建议先导出备份。

:::

## 最佳实践

### 配置管理

- **定期备份**：定期导出配置备份
- **版本控制**：将配置文件纳入版本控制
- **文档记录**：记录重要的配置更改

### 配置优化

- **按需配置**：只配置需要的选项
- **保持简洁**：避免过度复杂的配置
- **定期审查**：定期审查和更新配置

### 团队协作

- **共享配置**：与团队共享配置文件
- **配置规范**：建立团队配置规范
- **文档说明**：为配置提供文档说明
