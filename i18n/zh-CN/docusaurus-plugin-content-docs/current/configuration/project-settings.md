# 项目设置

项目设置允许您配置单个项目的特定行为和偏好。

## 功能概述

项目设置提供了以下功能：

- **项目信息管理**：编辑项目的基本信息
- **项目范围配置**：设置项目的可见性范围
- **项目通知**：配置项目相关的通知
- **项目成员**：管理项目成员和权限
- **项目集成**：配置第三方服务集成

## 打开项目设置

您可以通过以下方式打开项目设置：

1. 在项目列表中点击项目的设置图标
2. 打开项目详情，点击 **设置** 按钮
3. 右键点击项目，选择 **设置**

## 项目信息

### 基本信息

编辑项目的基本信息：

- **项目名称**：修改项目名称
- **项目描述**：修改项目描述
- **项目标签**：添加或修改项目标签
- **项目颜色**：设置项目的标识颜色

### 项目路径

- **路径显示**：显示项目的完整路径
- **路径修改**：修改项目路径（如果项目移动）
- **路径验证**：验证路径是否有效

### 项目类型

- **本地项目**：本地文件系统中的项目
- **远程项目**：连接到 Git 远程仓库的项目
- **项目类型转换**：在类型间转换（如果支持）

## 项目范围

### 可见性设置

配置项目在界面中的可见性：

- **全局范围**：项目在全局视图中可见
  - 适合：经常使用的项目
  - 效果：在所有视图的项目列表中显示

- **项目范围**：项目仅在项目视图中可见
  - 适合：不常用或特定用途的项目
  - 效果：仅在项目视图中显示

### 范围切换

切换项目范围的影响：

- **全局 → 项目**：项目从全局视图隐藏
- **项目 → 全局**：项目在全局视图显示
- **会话影响**：不影响现有会话

## 项目通知

### 通知类型

配置项目相关的通知：

- **会话通知**：项目中会话相关的通知
- **提案通知**：项目中提案相关的通知
- **更新通知**：项目更新的通知
- **错误通知**：项目错误和警告的通知

### 通知方式

选择接收通知的方式：

- **界面通知**：在界面中显示通知
- **桌面通知**：系统级桌面通知
- **邮件通知**：发送邮件通知
- **移动推送**：移动设备推送通知（如果支持）

### 通知过滤

过滤不需要的通知：

- **按类型过滤**：选择要接收的通知类型
- **按严重程度过滤**：选择要接收的严重程度
- **免打扰时段**：设置不接收通知的时段

## 项目成员

### 成员列表

查看项目的所有成员：

- **成员名称**：成员的显示名称
- **成员角色**：成员在项目中的角色
- **加入时间**：成员加入项目的时间
- **活跃状态**：成员的活跃状态

### 成员角色

项目成员有以下角色：

- **所有者**：项目的创建者和拥有者
  - 完全控制权
  - 可以删除项目
  - 可以管理所有成员

- **管理员**：项目的管理员
  - 管理项目设置
  - 管理普通成员
  - 不能删除项目

- **成员**：普通项目成员
  - 查看和编辑项目
  - 创建会话和提案
  - 不能修改项目设置

- **观察者**：只读访问
  - 仅查看项目
  - 不能创建或修改内容

### 邀请成员

邀请新成员加入项目：

1. 点击 **邀请成员** 按钮
2. 输入成员的邮箱或用户名
3. 选择成员角色
4. 添加欢迎消息（可选）
5. 发送邀请

### 管理成员

对现有成员进行管理：

- **更改角色**：提升或降低成员角色
- **移除成员**：从项目中移除成员
- **禁用成员**：暂时禁用成员访问

## 项目集成

### Git 集成

配置 Git 相关设置：

- **仓库 URL**：远程仓库的地址
- **认证方式**：SSH 或 HTTPS 认证
- **默认分支**：项目的默认分支
- **拉取策略**：自动或手动拉取

### CI/CD 集成

配置持续集成和部署：

- **CI 服务**：选择 CI 服务提供商
- **Webhook URL**：配置 Webhook 地址
- **自动部署**：配置自动部署策略
- **构建通知**：接收构建状态通知

### 外部服务

配置其他外部服务：

- **监控服务**：集成监控和日志服务
- **通知服务**：集成通知服务（如 Slack）
- **文档服务**：集成文档托管服务
- **项目管理**：集成项目管理工具

## 高级设置

### 性能设置

配置项目相关的性能选项：

- **并发限制**：项目的并发会话限制
- **缓存策略**：配置缓存行为
- **资源限制**：设置资源使用限制

### 安全设置

配置项目相关的安全选项：

- **访问控制**：配置访问控制策略
- **数据加密**：启用数据加密
- **审计日志**：启用操作审计

### 实验性功能

启用或禁用实验性功能：

- **功能列表**：显示可用的实验性功能
- **功能描述**：每个功能的描述
- **启用/禁用**：切换功能状态

## 项目删除

### 删除项目

从 PCode 中删除项目：

1. 打开项目设置
2. 滚动到底部
3. 点击 **删除项目**
4. 输入项目名称确认
5. 确认删除

:::caution 注意

- 删除项目不会删除文件系统中的项目文件
- 删除项目会删除所有相关的会话数据
- 此操作不可撤销，请谨慎操作

:::

### 归档项目

作为替代方案，可以归档项目：

- 保留项目数据和配置
- 隐藏项目列表
- 可以随时恢复

## 最佳实践

### 项目组织

- **清晰命名**：使用清晰、描述性的项目名称
- **适当范围**：合理设置项目范围
- **标签分类**：使用标签组织项目

### 成员管理

- **最小权限**：按需分配最小权限
- **定期审查**：定期审查成员列表
- **及时更新**：成员变动时及时更新

### 集成配置

- **安全第一**：使用安全的认证方式
- **监控配置**：配置适当的监控和告警
- **文档记录**：记录集成配置和流程
