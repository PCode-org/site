# Installation Guide

Welcome to the PCode installation guide. This page will help you get started with setting up PCode on your system.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** >= 18.0
- **npm** or **yarn** package manager

## Quick Start

### Install via npm

```bash
npm install pcode
```

### Install via yarn

```bash
yarn add pcode
```

## Next Steps

After installation, check out the [Feature Guides](/docs/feature-guides) to learn more about what you can do with PCode.

## Additional Configuration

For advanced configuration options, please refer to the official documentation.

## Troubleshooting

If you encounter any issues during installation, please:

1. Check that your Node.js version meets the requirements
2. Clear your npm cache: `npm cache clean --force`
3. Try reinstalling with the `--legacy-peer-deps` flag if using npm
