# User Guide

Welcome to the PCode User Guide. This guide will help you get started and make the most of PCode's features.

## Quick Start

If you're new to PCode, we recommend reading in this order:

1. **[Appearance - Themes](./appearance/themes)** - Choose a theme that suits you
2. **[Appearance - Languages](./appearance/languages)** - Choose your preferred language
3. **[Settings Overview](./settings/)** - Learn about available settings

### Appearance

Customize PCode's appearance:

- **[Themes](./appearance/themes)** - Switch between interface themes

### Settings

Manage PCode settings:
