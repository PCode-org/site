## 🛠️ 文档站点开发

基于 <a href="https://docusaurus.io/">Docusaurus</a> 构建的文档站点

<table>
<tr>
<td width="50%">

<h4>快速开始</h4>

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm build
```

</td>
<td width="50%">

<h4>环境要求</h4>

| 依赖 | 版本 |
|:----|:-----|
| Node.js | ≥18.0 |

<h4>项目结构</h4>

```
pcode-docs/
├── docs/          # 文档内容
│   ├── installation/
│   ├── quick-start/
│   └── ...
├── src/           # 源代码
│   ├── components/
│   ├── css/
│   └── pages/
├── static/        # 静态资源
├── docusaurus.config.ts
└── sidebars.ts
```
